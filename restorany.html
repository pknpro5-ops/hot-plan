Специализированные решения для HoReCa: кухонные вытяжки, жироуловители, расчет по СанПиН. Опыт работы с Кофеманией и Мореманией.

Особенности проектирования

  • Раздельная вентиляция

    Проектируем независимые системы для горячего цеха и обеденного зала согласно СП 60.13330.

  • Жироуловители

    Подбор и расчет систем очистки стоков для согласования с Мосводоканалом.

  • Гидрофильтры

    Интеграция систем искрогашения для зон с открытым огнем (хосперы, мангалы).
